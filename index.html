Savetek.js

A lightweight JavaScript library for easy interaction with the localStorage API.

Get Started

Installation

You can include Savetek.js in your project by either downloading it or linking directly to the CDN.

<script src="https://cdn.jsdelivr.net/npm/savetek@1.0.2/savetek.min.js"></script>

Usage

Savetek.js provides a simple and intuitive interface for localStorage operations.

SaveTek.set('key', 'value');
SaveTek.get('key');
SaveTek.remove('key');

Examples

Here are some examples demonstrating how to use Savetek.js in various scenarios.

// Set a value in localStorage
SaveTek.set('username', 'john_doe');

// Retrieve a value from localStorage
const username = SaveTek.get('username');
console.log(username); // Output: john_doe

// Remove a value from localStorage
SaveTek.remove('username');
